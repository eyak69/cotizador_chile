# Documentación de Diseños (Stitch Mockups)

Esta carpeta almacena los mockups (diseños) generados por la IA (Stitch UI) que se utilizaron como base para aplicar el rediseño **Dark Mode Premium** a Cotizador IA.

## Pantallas Generadas:

### 1. Dashboard Principal (Insurance Dashboard Premium Dark)
*   **Dispositivo:** Móvil (Adaptado a Desktop)
*   **Enfoque:** Tarjetas Glassmorphism con bordes ahumados y listados modernos.
*   **Archivo:** `mockup_dashboard_premium.jpg`

### 2. Panel de Cotización (Nueva Cotización Step 2: Vehículo)
*   **Dispositivo:** Móvil (Adaptado a Desktop)
*   **Enfoque:** Inputs de texto limpios con botones redondeados en gradientes morado/azul.
*   **Archivo:** `mockup_cotizacion_vehiculo.jpg`

## Detalles Técnicos
*   **Proyecto 1 (Cotizador de Seguros):** `14959302658120890447`
*   **Proyecto 2 (Dark Glassmorphism Dashboard):** `9628937010815254960`
*   El diseño se aplicó en React (Vite) utilizando clases utilitarias de CSS puro en `index.css` (`.glass-card`, `.glow-border`, variables CSS), con fondos globales aplicados a través del componente `CssBaseline` de Material UI. Todo este trabajo se estructuró a partir de las paletas y degradados que originó la inteligencia artificial de programación visual.
